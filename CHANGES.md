# lein-miditest changelog

## 0.1.0 [`tag`][0.1.0-tag]

* Fallbacks to the default instrument if it is unable to find the default
  success and failure instrument.
* Plays a french horn on failure, and a timpani on success. Not configurable.
* First release!
